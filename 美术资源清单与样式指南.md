# 《微恐咖啡厅》美术资源清单与样式指南

## 一、美术风格规范

### 1.1 整体风格
- **艺术风格**: 卡通手绘 + 轻度恐怖元素
- **色彩基调**: 暗色调为主，亮色点缀
- **细节处理**: 简洁线条，避免过度细节
- **恐怖程度**: PG-13级，适合大众

### 1.2 恐怖元素处理
```
允许的元素:
- 幽灵轮廓 (半透明，可爱化)
- 蜘蛛网装饰 (卡通风格)
- 蝙蝠剪影
- 南瓜灯 (微笑表情)
- 诡异光线效果

禁止的元素:
- 血腥暴力
- 惊吓跳脸
- 恐怖音效
- 阴暗压抑场景
```

## 三、资源优化方案

### 3.1 图片优化策略
```
1. 格式选择
   - UI元素: PNG-8 (256色) 或 WebP
   - 背景图: JPG (质量85%)
   - 序列帧: PNG序列 → 雪碧图

2. 尺寸压缩
   - 设计稿750px → 实际375px (@2x适配)
   - 最大单图不超过512px
   - 小图标使用CSS绘制替代

3. 雪碧图打包
   - 按钮状态打包
   - 图标集合打包
   - 动画序列打包
```

### 3.2 加载策略
```
1. 首屏资源 (<2MB)
   - 主界面背景
   - 基础UI元素
   - 核心按钮图标

2. 按需加载
   - 子界面资源进入时加载
   - 角色动画使用时加载
   - 音效首次播放时加载

3. 缓存策略
   - 常驻资源常驻内存
   - 使用频率低的资源及时释放
   - 微信小游戏本地缓存
```

### 3.3 性能监控指标
```
目标性能指标:
- 首屏加载时间: <3秒
- 界面切换时间: <1秒
- 内存使用: <100MB
- 帧率: 稳定60fps
- 包体大小: <8MB
```
