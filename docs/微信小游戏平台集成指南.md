## 🔧 配置步骤

### 步骤1：微信开发者账号准备
1. **注册微信开放平台账号**

### 步骤3：微信API适配集成

#### 在游戏启动时初始化
```typescript
// 在游戏启动脚本（如main.ts）中添加
import { WeChatPlatformAdapter } from './WeChatPlatformAdapter';

export function initializeGame() {
    // 初始化游戏逻辑
    const game = new Game();
    
    // 初始化微信平台适配
    WeChatPlatformAdapter.initWeChatAPI();
    
    // 注册微信登录回调
    WeChatPlatformAdapter.onLogin((userInfo) => {
        console.log('用户登录成功:', userInfo);
        game.onUserLogin(userInfo);
    });
    
    return game;
}
```

## 🔍 调试与排错

### 常见问题及解决方案

#### 问题1：构建失败
**症状**: Cocos Creator构建报错
**解决方案**:
```bash
# 清理缓存
rm -rf ./library
rm -rf ./temp

# 重新构建
npm run build:wechat
```

#### 问题2：微信API调用失败
**症状**: `wx is not defined`
**解决方案**:
```javascript
// 检查环境
if (typeof wx === 'undefined') {
    console.warn('⚠️ 非微信环境，启用模拟API');
    // 加载模拟API
    require('./mock/wx-mock.js');
}
```

#### 问题3：资源加载失败
**症状**: 图片或声音无法加载
**解决方案**:
```javascript
// 使用微信资源加载
wx.loadSubpackage({
    name: 'assets',
    success: () => {
        console.log('✅ 资源分包加载成功');
    },
    fail: (err) => {
        console.error('❌ 资源加载失败:', err);
    }
});
```

#### 问题4：性能问题
**症状**: 游戏卡顿或崩溃
**解决方案**:
1. 使用分包加载减少首包体积
2. 使用纹理压缩减少内存占用
3. 优化JavaScript代码，避免频繁GC
4. 使用对象池复用游戏对象

## 🔄 更新维护

### 1. 热更新
微信小游戏支持资源热更新：

```javascript
// 检查更新
const updateManager = wx.getUpdateManager();

updateManager.onCheckForUpdate((res) => {
    if (res.hasUpdate) {
        console.log('发现新版本，开始更新...');
    }
});

updateManager.onUpdateReady(() => {
    wx.showModal({
        title: '更新提示',
        content: '新版本已准备好，是否重启应用？',
        success: (res) => {
            if (res.confirm) {
                updateManager.applyUpdate();
            }
        }
    });
});
```
