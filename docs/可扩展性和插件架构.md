# 可扩展性和插件架构文档

## 概述

agency_core.py 性能优化版本设计为高度可扩展的微型多智能体框架。通过插件化架构、模块化设计和清晰的接口定义，系统可以轻松扩展新功能。

## 架构设计原则

### 1. 单一职责原则
每个组件只负责一项核心功能：
- `BaseAgent`: 智能体核心逻辑
- `AsyncTaskPool`: 异步任务管理
- `MemoryCache`: 内存缓存
- `PerformanceMonitor`: 性能监控
- `AgentLogger`: 日志管理
- `ErrorHandler`: 错误处理

### 2. 开闭原则
系统对扩展开放，对修改封闭：
- 通过继承添加新智能体类型
- 通过插件添加新功能
- 通过配置文件调整参数

### 3. 依赖倒置原则
高层模块不依赖低层模块，二者都依赖抽象：
- 定义清晰的接口规范
- 通过依赖注入配置组件
- 支持多种实现替换

### 插件接口定义

```python
class PluginInterface(ABC):
    """插件基础接口"""
    
    @abstractmethod
    def initialize(self, config: Dict[str, Any]) -> bool:
        """插件初始化"""
        pass
    
    @abstractmethod
    def process(self, data: Any) -> Any:
        """处理数据"""
        pass
    
    @abstractmethod
    def shutdown(self) -> bool:
        """插件关闭"""
        pass
    
    @abstractmethod
    def get_metadata(self) -> Dict[str, Any]:
        """获取插件元数据"""
        pass
```

### 插件管理器

```python
class PluginManager:
    """插件管理器"""
    
    def __init__(self):
        self.plugins = {}
        self.plugin_configs = {}
    
    def register_plugin(self, plugin_name: str, plugin_class: Type[PluginInterface], 
                       config: Dict[str, Any] = None):
        """注册插件"""
        self.plugin_configs[plugin_name] = config or {}
        print(f"[插件管理器] 注册插件: {plugin_name}")
    
    def load_plugin(self, plugin_name: str, **kwargs) -> PluginInterface:
        """加载插件实例"""
        if plugin_name not in self.plugin_configs:
            raise ValueError(f"插件 {plugin_name} 未注册")
        
        config = self.plugin_configs[plugin_name].copy()
        config.update(kwargs)
        
        # 动态加载插件
        try:
            plugin = self._load_plugin_class(plugin_name)(**config)
            if plugin.initialize(config):
                self.plugins[plugin_name] = plugin
                print(f"[插件管理器] 加载插件成功: {plugin_name}")
                return plugin
            else:
                raise RuntimeError(f"插件初始化失败: {plugin_name}")
        except Exception as e:
            raise RuntimeError(f"加载插件失败 {plugin_name}: {str(e)}")
    
    def _load_plugin_class(self, plugin_name: str) -> Type[PluginInterface]:
        """动态加载插件类"""
        # 这里可以支持从文件系统、数据库等加载插件
        # 示例：从 plugins 目录加载
        plugin_file = f"plugins/{plugin_name}.py"
        if os.path.exists(plugin_file):
            return self._import_plugin_from_file(plugin_file, plugin_name)
        else:
            # 内置插件
            return BUILTIN_PLUGINS.get(plugin_name)
    
    def unload_plugin(self, plugin_name: str) -> bool:
        """卸载插件"""
        if plugin_name in self.plugins:
            plugin = self.plugins[plugin_name]
            if plugin.shutdown():
                del self.plugins[plugin_name]
                print(f"[插件管理器] 卸载插件: {plugin_name}")
                return True
        return False
    
    def get_plugin(self, plugin_name: str) -> Optional[PluginInterface]:
        """获取插件实例"""
        return self.plugins.get(plugin_name)
    
    def list_plugins(self) -> List[Dict[str, Any]]:
        """列出所有插件"""
        result = []
        for name, plugin in self.plugins.items():
            result.append({
                "name": name,
                "metadata": plugin.get_metadata(),
                "config": self.plugin_configs.get(name, {})
            })
        return result
```

#### 钩子方法 (Hooks)
```python
class ExtensibleBaseAgent(BaseAgent):
    """可扩展的智能体基类"""
    
    def __init__(self, *args, **kwargs):
        super().__init__(*args, **kwargs)
        self.pre_execute_hooks = []
        self.post_execute_hooks = []
        self.validation_hooks = []
    
    def add_pre_execute_hook(self, hook: Callable):
        """添加执行前钩子"""
        self.pre_execute_hooks.append(hook)
    
    def add_post_execute_hook(self, hook: Callable):
        """添加执行后钩子"""
        self.post_execute_hooks.append(hook)
    
    def execute_task(self, task_description: str, max_retries: int = 3) -> Dict[str, Any]:
        """增强的执行任务方法，支持钩子"""
        
        # 执行前钩子
        for hook in self.pre_execute_hooks:
            modified_task = hook(task_description)
            if modified_task is not None:
                task_description = modified_task
        
        # 执行原始逻辑
        result = super().execute_task(task_description, max_retries)
        
        # 执行后钩子
        for hook in self.post_execute_hooks:
            modified_result = hook(result)
            if modified_result is not None:
                result = modified_result
        
        return result
```

## 配置系统设计

### 分层配置

```python
class ConfigManager:
    """配置管理器"""
    
    def __init__(self):
        self.config = {
            "global": {},      # 全局配置
            "agent": {},       # 智能体配置
            "plugin": {},      # 插件配置
            "performance": {}, # 性能配置
            "storage": {}      # 存储配置
        }
    
    def load_config(self, config_file: str):
        """从文件加载配置"""
        if os.path.exists(config_file):
            with open(config_file, "r", encoding="utf-8") as f:
                file_config = json.load(f)
                self._merge_config(file_config)
    
    def _merge_config(self, new_config: Dict[str, Any]):
        """合并配置"""
        for key, value in new_config.items():
            if isinstance(value, dict) and key in self.config:
                self.config[key].update(value)
            else:
                self.config[key] = value
    
    def get_config(self, section: str, key: str = None, default: Any = None):
        """获取配置"""
        if section not in self.config:
            return default
        
        if key is None:
            return self.config[section]
        
        return self.config[section].get(key, default)
```

## 事件系统设计

### 事件总线

```python
class EventBus:
    """事件总线"""
    
    def __init__(self):
        self.subscribers = defaultdict(list)
        self.event_history = deque(maxlen=1000)
    
    def subscribe(self, event_type: str, callback: Callable):
        """订阅事件"""
        self.subscribers[event_type].append(callback)
    
    def unsubscribe(self, event_type: str, callback: Callable):
        """取消订阅"""
        if callback in self.subscribers[event_type]:
            self.subscribers[event_type].remove(callback)
    
    def publish(self, event_type: str, data: Any):
        """发布事件"""
        event = {
            "type": event_type,
            "data": data,
            "timestamp": time.time()
        }
        self.event_history.append(event)
        
        # 通知订阅者
        for callback in self.subscribers.get(event_type, []):
            try:
                callback(event)
            except Exception as e:
                print(f"[事件总线] 回调执行失败: {str(e)}")
    
    def get_recent_events(self, event_type: Optional[str] = None, limit: int = 100):
        """获取最近事件"""
        events = list(self.event_history)
        if event_type:
            events = [e for e in events if e["type"] == event_type]
        return events[-limit:]
```

```python
class RedisCachePlugin(StoragePlugin):
    """Redis缓存插件"""
    
    def __init__(self, host: str = "localhost", port: int = 6379, db: int = 0):
        import redis
        self.client = redis.Redis(host=host, port=port, db=db, decode_responses=True)
    
    def initialize(self, config: Dict[str, Any]) -> bool:
        try:
            self.client.ping()
            print("[Redis缓存插件] 连接成功")
            return True
        except Exception as e:
            print(f"[Redis缓存插件] 连接失败: {str(e)}")
            return False
    
    def save(self, key: str, data: Any) -> bool:
        try:
            self.client.set(key, json.dumps(data), ex=3600)  # 1小时过期
            return True
        except Exception as e:
            print(f"[Redis缓存插件] 保存失败: {str(e)}")
            return False
    
    def load(self, key: str) -> Optional[Any]:
        try:
            data = self.client.get(key)
            if data:
                return json.loads(data)
        except Exception as e:
            print(f"[Redis缓存插件] 加载失败: {str(e)}")
        return None
    
    def shutdown(self) -> bool:
        self.client.close()
        print("[Redis缓存插件] 关闭连接")
        return True
    
    def get_metadata(self) -> Dict[str, Any]:
        return {
            "name": "RedisCachePlugin",
            "version": "1.0.0",
            "storage_type": "redis",
            "connected": self.client.ping() if hasattr(self, 'client') else False
        }
```

## 性能调优建议

### 1. 内存优化
- 使用适当的内存窗口大小
- 启用缓存并设置合理的过期时间
- 定期清理已完成的任务
- 使用内存分析工具（如memory_profiler）

### 2. CPU优化
- 调整并发任务数（max_concurrent_tasks）
- 使用线程池而不是进程池
- 避免CPU密集型操作在主线程中执行
- 使用异步I/O操作

### 3. 网络优化
- 启用API响应缓存
- 使用连接池管理数据库连接
- 实现请求重试和退避机制
- 监控网络延迟和带宽

### 4. 存储优化
- 选择合适的存储后端（根据数据量）
- 实现数据分片和分区
- 使用索引加速查询
- 定期备份和归档旧数据

## 故障排除

### 常见问题及解决方案

1. **内存泄漏**
   - 检查缓存设置和清理策略
   - 确保所有资源正确释放
   - 使用gc.collect()强制垃圾回收

2. **性能下降**
   - 监控系统资源使用情况
   - 检查任务执行时间分布
   - 优化复杂任务的拆分

3. **插件加载失败**
   - 检查插件依赖是否满足
   - 验证插件配置参数
   - 查看插件初始化日志

4. **配置错误**
   - 使用配置验证工具
   - 检查环境变量设置
   - 验证配置文件格式

## 总结

agency_core.py 性能优化版本通过插件化架构、模块化设计和清晰的扩展点，提供了强大的可扩展性。开发者可以通过：

1. **实现插件接口**来添加新功能
2. **继承基类**来定制智能体行为
3. **配置系统**来调整运行时参数
4. **事件系统**来实现松耦合集成

这种设计使得系统能够适应不同的业务需求和技术栈，同时保持核心逻辑的稳定性和可维护性。
