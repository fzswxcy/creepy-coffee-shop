# 微信小游戏真实广告配置指南

## 📋 广告单元ID获取步骤

### 1. 注册微信广告平台
**前提条件：**
- 已注册微信小程序/小游戏账号
- 已完成小游戏主体认证

**注册流程：**
1. 访问 https://ad.weixin.qq.com
2. 使用小程序/小游戏管理员账号登录
3. 完成广告主资质认证
4. 创建广告应用

### 2. 创建广告位（广告单元）

#### 激励视频广告
**创建位置：** 微信广告平台 → 流量主 → 广告位管理 → 新建广告位

**配置建议：**
- **广告位类型**: 激励视频广告
- **广告位名称**: 微恐咖啡厅_激励视频_主按钮
- **广告场景**: 游戏内奖励
- **用户激励描述**: "观看广告获得双倍金币奖励"
- **用户激励时长**: 15-30秒
- **广告样式**: 原生激励视频

**获取ID格式：**
```
adunit-xxxxx-rewarded-video
```

#### 插屏广告
**配置建议：**
- **广告位类型**: 插屏广告
- **广告位名称**: 微恐咖啡厅_插屏_游戏间隙
- **广告场景**: 游戏场景切换
- **展示时机**: 游戏暂停、关卡结束、返回主菜单
- **关闭按钮延迟**: 3-5秒

**获取ID格式：**
```
adunit-xxxxx-interstitial
```

#### 横幅广告（可选）
**配置建议：**
- **广告位类型**: 横幅广告
- **广告位名称**: 微恐咖啡厅_横幅_底部
- **广告场景**: 游戏主界面底部
- **广告尺寸**: 360×160px（微信推荐）
- **刷新间隔**: 30秒

**获取ID格式：**
```
adunit-xxxxx-banner
```

### 4. 多环境配置管理

#### 方案一：环境变量配置
```typescript
// config/AdConfig.ts
export class AdConfig {
    static get adUnitIds() {
        const env = process.env.NODE_ENV || 'development';
        
        if (env === 'production') {
            return {
                rewardedVideo: 'adunit-real-rewarded-video-id',
                interstitial: 'adunit-real-interstitial-id',
                banner: 'adunit-real-banner-id'
            };
        } else {
            return {
                rewardedVideo: 'adunit-test-rewarded-video',
                interstitial: 'adunit-test-interstitial',
                banner: 'adunit-test-banner'
            };
        }
    }
}
```

#### 方案二：配置文件切换
```typescript
// config/AdConfig.ts
import { AD_UNIT_IDS_TEST } from './AdConfig.test';
import { AD_UNIT_IDS_PROD } from './AdConfig.prod';

const IS_PROD = window.location.hostname !== 'localhost';

export const AD_UNIT_IDS = IS_PROD ? AD_UNIT_IDS_PROD : AD_UNIT_IDS_TEST;
```

### 6. 广告审核注意事项

#### 激励视频广告审核要点
1. **激励明确性**: 必须明确告知用户观看广告后获得的奖励
2. **自愿原则**: 用户必须自愿选择观看广告
3. **奖励及时性**: 奖励必须在广告完成后立即发放
4. **广告质量**: 广告内容必须符合微信平台规范

#### 插屏广告审核要点
1. **展示时机**: 不能打断用户的核心游戏操作
2. **关闭按钮**: 必须有明显的关闭按钮
3. **频率控制**: 不能过于频繁展示
4. **内容合规**: 广告内容必须符合法律法规

### 8. 收益优化建议

#### 提高填充率
1. **多渠道广告源**: 集成多家广告平台
2. **广告样式优化**: 使用平台推荐的最佳实践
3. **用户标签优化**: 根据用户行为展示相关广告

#### 提高eCPM
1. **广告场景优化**: 在用户活跃度高的场景展示广告
2. **用户分层**: 针对高价值用户展示高价广告
3. **季节性调整**: 根据节日调整广告策略

#### 提高完成率
1. **奖励价值优化**: 设置合适的奖励价值
2. **等待时间优化**: 减少广告加载等待时间
3. **用户体验优化**: 确保广告流程顺畅

### 10. 合规和安全

#### 合规要求
1. **用户隐私**: 遵守《个人信息保护法》
2. **未成年人保护**: 限制未成年人广告展示
3. **广告内容**: 符合广告法规定
4. **数据安全**: 广告数据传输加密

#### 安全措施
1. **防作弊机制**: 防止虚假广告点击
2. **数据验证**: 广告数据真实性验证
3. **异常监控**: 实时监控异常广告行为
4. **审计日志**: 保留完整的操作日志

## 🚨 常见问题和解决方案

### 问题1：广告加载失败
**可能原因：**
- 广告单元ID错误
- 网络连接问题
- 微信SDK版本过低

**解决方案：**
1. 检查广告单元ID是否正确
2. 添加网络重试机制
3. 使用降级方案（模拟广告）

### 问题2：广告填充率低
**可能原因：**
- 新应用广告资源不足
- 用户标签不准确
- 广告请求频率过高

**解决方案：**
1. 集成多家广告平台
2. 优化用户画像
3. 调整广告请求策略

### 问题3：收益低于预期
**可能原因：**
- eCPM设置不合理
- 用户质量不高
- 广告展示时机不佳

**解决方案：**
1. A/B测试优化eCPM
2. 提高用户留存和活跃度
3. 优化广告展示场景

### 问题4：用户投诉广告过多
**可能原因：**
- 广告频率设置过高
- 广告展示时机不当
- 奖励价值不合理

**解决方案：**
1. 调整广告频率限制
2. 优化广告展示时机算法
3. 合理设置奖励价值
