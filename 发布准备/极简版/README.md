## 🔧 快速部署步骤

### 步骤1：准备环境
```bash
# 确保已安装
# 1. Cocos Creator 3.8+
# 2. Node.js 14+
# 3. npm

# 进入项目目录
cd /root/.openclaw/workspace/微恐咖啡厅
```

### 步骤2：执行一键部署
```bash
# 给脚本执行权限
chmod +x 发布准备/极简版/build-and-deploy.sh

# 执行部署脚本
./发布准备/极简版/build-and-deploy.sh
```

### 步骤3：检查构建结果
脚本会自动：
1. ✅ 检查环境依赖
2. ✅ 清理旧构建文件
3. ✅ 安装项目依赖
4. ✅ 构建微信小游戏
5. ✅ 检查构建质量
6. ✅ 创建部署包
7. ✅ 生成部署指南

### 步骤4：导入微信开发者工具
1. 打开微信开发者工具
2. 点击"导入项目"
3. 选择：`./build-quick/wechatgame-quick`
4. 输入你的AppID
5. 点击"导入"

### 步骤5：测试运行
1. 点击"编译"按钮
2. 在模拟器中测试游戏
3. 点击"预览"生成二维码
4. 微信扫码真机测试

### 步骤6：提交审核
1. 点击"上传"按钮
2. 填写版本信息
3. 提交到微信平台
4. 等待审核（1-3天）

**重要**：需要将 `wechat-project.config.json` 中的 AppID 替换为你的实际微信小游戏AppID。

### 2. 微信审核要求

- 需要有游戏介绍和截图
