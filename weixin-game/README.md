# 微恐咖啡厅 - 微信小游戏项目

## 🎮 项目概述
**项目名称**: 微恐咖啡厅  
**平台**: 微信小游戏 (手机端)  
**技术栈**: Cocos Creator 3.8+ (微信小游戏兼容模式)  
**核心玩法**: 午夜咖啡厅经营 + 恐怖解谜 + 时间循环


## 🚀 核心技术特性

### 1. 微信小游戏原生集成
- ✅ 微信登录系统

- **理智值管理**: 随时间下降，影响恐怖事件频率
- **咖啡制作系统**: 触摸屏交互，多种配方
- **恐怖事件系统**: 随机触发，影响游戏体验
- **成就系统**: 解锁奖励，提高用户粘性
- **时间循环机制**: 核心解谜玩法

### 3. 性能优化
- **Canvas 2D渲染**: 轻量级，微信小游戏友好
- **内存管理**: 严格控制资源使用
- **FPS监控**: 实时性能监控
- **触摸优化**: 针对手机屏幕的交互设计

## 🎯 游戏玩法

### 核心循环
1. **制作咖啡**: 触摸咖啡机，消耗理智值获得金币
2. **保持理智**: 清洁吧台恢复理智值
3. **应对恐怖**: 随机恐怖事件，需要快速反应
4. **解锁配方**: 随着游戏进展解锁新咖啡配方
5. **时间循环**: 每24小时重置，但保留进度

### 恐怖事件类型
- 幽灵闪现
- 灯光闪烁  
- 奇怪声音
- 咖啡杯移动
- 温度骤降

## 🚨 技术限制与挑战

### 微信小游戏限制
- **包体积**: 初始包≤4MB，总包≤20MB
- **内存**: 运行内存≤2048MB
- **网络**: 域名白名单限制
- **API**: 部分Web API不可用

### 性能优化重点
1. **图片压缩**: 使用webp格式，控制尺寸
2. **代码分包**: 按需加载，减少初始包体积
3. **内存回收**: 及时销毁无用对象
4. **渲染优化**: 减少重绘，使用离屏Canvas
