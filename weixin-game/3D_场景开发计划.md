## 🎨 3D场景资源清单

### 1. 核心场景物体
#### 必须创建的3D模型
- **吧台** (Counter) - 4x1.5x1 米，深色木质
- **咖啡机** (CoffeeMachine) - 0.8x1.2x0.8 米，金属质感
- **桌子** (Table) x5 - 0.8x0.6x0.8 米，浅色木质
- **椅子** (Chair) x5 - 0.4x0.8x0.4 米，深色木质
- **咖啡杯** (CoffeeCup) - 0.2x0.3x0.2 米，陶瓷质感
- **咖啡豆袋** (CoffeeBeanBag) - 0.4x0.5x0.3 米，布袋质感
- **时钟** (Clock) - 0.5x0.5x0.1 米，金属边框
- **海报** (Poster) - 1.2x1.6x0.05 米，老旧纸张

### 2. 灯光系统
#### 必须实现的灯光效果
- **主灯光** - 点光源，昏暗黄色，位置(0,3,-1)
- **吧台灯光** - 聚光灯，暖白色，位置(0,2.5,-2.8)
- **环境光** - 全局环境光，低强度
- **红色警示灯** - 恐怖模式，位置(2,2,-4)
- **蓝色幽灵光** - 幽灵区域，位置(-2,1.5,-4)

### 3. 恐怖特效系统
#### 核心恐怖氛围元素
- **幽灵模型** - 半透明，漂浮移动
- **血滴效果** - 墙面滴落动画
- **雾气效果** - 环境雾气，随理智值变化
- **闪烁特效** - 灯光和物体闪烁
- **屏幕特效** - 震动、模糊、色相变化

### 4. 材质系统
#### 必须创建的材质类型
- **wood_dark** - 深色木质 (吧台、椅子)
- **wood_light** - 浅色木质 (桌子)
- **metal** - 金属质感 (咖啡机、时钟)
- **porcelain** - 陶瓷质感 (咖啡杯)
- **fabric** - 布料质感 (咖啡豆袋)
- **paper** - 纸张质感 (海报)
- **ghost** - 幽灵半透明材质
- **horror** - 恐怖特效材质

## 🎯 质量验收标准

### 技术标准
1. **性能**: FPS ≥ 30（低端手机）
2. **内存**: ≤ 512MB（微信小游戏限制）
3. **包体积**: 初始包 ≤ 4MB，总包 ≤ 20MB
4. **兼容性**: 支持iOS/Android微信

### 视觉效果标准
1. **恐怖氛围**: 有足够的恐怖元素和氛围
2. **美术风格**: 统一的中式恐怖+咖啡厅风格
3. **动画流畅**: 所有动画流畅自然
4. **灯光效果**: 灯光系统营造恐怖氛围

### 用户体验标准
1. **交互响应**: 触摸响应时间 ≤ 100ms
2. **操作便捷**: 单手可完成所有操作
3. **引导清晰**: 新用户容易上手
4. **性能稳定**: 无崩溃和明显卡顿

- 测试多种手机型号

### 风险3：包体积超标
**应对**:
- 使用纹理压缩
- 实现资源分包加载
- 移除不必要资源

- 优先核心功能开发
- 使用预制资源和模板
