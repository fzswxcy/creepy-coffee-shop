## 🐛 常见问题解决

### 问题1: Cocos Creator环境问题
**症状**: 构建脚本失败，提示Cocos未找到
**解决方案**:
```bash
# 方法1: 设置环境变量
export PATH=$PATH:/Applications/CocosCreator.app/Contents/MacOS

# 方法2: 使用绝对路径调用
/Applications/CocosCreator.app/Contents/MacOS/cocos compile -p wechatgame
```

### 问题2: 微信开发者工具导入失败
**症状**: 导入项目时报错
**解决方案**:
1. 确认项目路径正确
2. 检查AppID配置
3. 清空开发者工具缓存
4. 重启开发者工具

### 问题3: 广告功能测试失败
**症状**: 广告无法正常显示
**解决方案**:
1. 确认广告位ID正确
2. 检查网络连接
3. 使用测试广告位ID
4. 查看控制台错误信息

### 问题4: 云数据库连接失败
**症状**: 数据无法保存/读取
**解决方案**:
1. 确认云环境初始化
2. 检查数据库权限
3. 验证网络连接
4. 查看云函数日志
