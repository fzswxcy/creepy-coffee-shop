# 广告展示功能测试用例

## 测试概述
测试微信小游戏广告系统的所有功能，包括激励视频、插屏广告、Banner广告等，确保广告展示、回调、收益计算等流程正常工作。

## 测试用例列表

### TC-ADS-001: 激励视频广告触发展示
- **测试场景**: 玩家点击激励视频广告入口
- **前置条件**:
  1. 游戏已正常启动
  2. 广告组件初始化成功
  3. 网络连接正常
  4. 用户未屏蔽广告
- **测试步骤**:
  1. 点击激励视频广告入口（如"双倍收益"按钮）
  2. 观察广告加载过程
  3. 等待广告完整播放
  4. 点击广告关闭按钮
- **预期结果**:
  1. 广告正常加载并展示
  2. 广告播放完整
  3. 关闭后触发回调函数
  4. 玩家获得相应奖励
- **优先级**: P0
- **测试类型**: 手动测试 + 真实广告环境测试

## 自动化测试脚本示例

```javascript
// 广告测试自动化脚本
const adTestingFramework = {
  // 初始化广告测试环境
  setupAdTestEnvironment: async (config) => {
    await setupNetworkConditions(config.network);
    await setupAdMock(config.adType);
    await clearAdCache();
  },
  
  // 测试激励视频广告
  testRewardedVideoAd: async (adPosition) => {
    const testResults = {
      loadTime: 0,
      playStatus: '',
      rewardGiven: false,
      errors: []
    };
    
    try {
      // 记录加载开始时间
      const loadStart = Date.now();
      
      // 触发广告
      await clickElement(adPosition.button);
      
      // 等待广告加载
      const adLoaded = await waitForAdLoad(10000);
      testResults.loadTime = Date.now() - loadStart;
      
      if (!adLoaded) {
        testResults.errors.push('广告加载超时');
        return testResults;
      }
      
      // 模拟广告播放
      await simulateAdPlayback();
      
      // 验证回调触发
      const callbackFired = await verifyAdCallback();
      testResults.playStatus = callbackFired ? 'completed' : 'failed';
      
      // 验证奖励发放
      if (callbackFired) {
        const rewardVerified = await verifyRewardGranted(adPosition.expectedReward);
        testResults.rewardGiven = rewardVerified;
      }
      
    } catch (error) {
      testResults.errors.push(error.message);
    }
    
    return testResults;
  },
  
  // 测试插屏广告频率控制
  testInterstitialFrequency: async (intervalSetting) => {
    const results = [];
    const triggerTimes = 10;
    
    for (let i = 0; i < triggerTimes; i++) {
      // 触发广告展示条件
      await triggerAdCondition();
      
      // 记录广告展示情况
      const adShown = await checkIfAdShown();
      results.push({
        attempt: i + 1,
        timestamp: Date.now(),
        adShown: adShown,
        timeSinceLast: i > 0 ? Date.now() - results[i-1].timestamp : 0
      });
      
      // 等待一段时间
      await wait(1000);
    }
    
    return results;
  },
  
  // 验证广告收益计算
  verifyAdRevenue: async (testSession) => {
    const clientData = await collectClientAdData();
    const serverData = await fetchServerRevenueData();
    
    const discrepancies = [];
    
    // 对比激励视频数据
    compareDataSets(
      clientData.rewardedVideos,
      serverData.rewardedVideos,
      ['adUnit', 'viewCount', 'revenue'],
      discrepancies
    );
    
    // 对比插屏广告数据
    compareDataSets(
      clientData.interstitials,
      serverData.interstitials,
      ['adUnit', 'impressions', 'revenue'],
      discrepancies
    );
    
    return {
      matchRate: (1 - discrepancies.length / 
        (clientData.rewardedVideos.length + clientData.interstitials.length)) * 100,
      discrepancies: discrepancies,
      totalRevenue: serverData.totalRevenue
    };
  }
};

module.exports = adTestingFramework;
```

观看完整激励视频广告后，奖励未正确发放到玩家账户。
