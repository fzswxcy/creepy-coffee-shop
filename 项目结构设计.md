## 📦 文件说明

### 核心配置文件

1. **package.json** - 项目依赖管理
   ```json
   {
     "name": "wechat-coffee-horror",
     "version": "1.0.0",
     "description": "微恐咖啡厅微信小游戏",
     "dependencies": {},
     "devDependencies": {}
   }
   ```

2. **project.json** - Cocos Creator项目配置
   ```json
   {
     "engine": "cocos-creator-3.x",
     "designResolution": {
       "width": 750,
       "height": 1334,
       "fitWidth": true,
       "fitHeight": false
     },
     "startScene": "db://assets/scenes/MainScene.fire",
     "platform": "wechatgame",
     "packages": {}
   }
   ```

3. **tsconfig.json** - TypeScript编译配置
   ```json
   {
     "compilerOptions": {
       "target": "es2015",
       "module": "commonjs",
       "lib": ["dom", "es2015"],
       "experimentalDecorators": true,
       "skipLibCheck": true,
       "outDir": "./temp",
       "strict": true,
       "noImplicitAny": false
     }
   }
   ```

## 🎯 包体控制策略

### 资源优化
1. **精灵图集**：合并小图，减少drawcall
2. **音频压缩**：使用MP3/WAV格式，采样率降低
3. **纹理压缩**：使用PVR/ETC压缩格式
4. **代码压缩**：UglifyJS + 代码分包

### 首次加载优化
1. **资源分包**：基础包<2MB，按需加载
2. **懒加载**：场景资源按需加载
3. **预加载**：核心资源预加载
4. **CDN加速**：静态资源使用CDN

## 🔧 开发环境配置

### 必备软件
1. **Cocos Creator** 3.8+
2. **Node.js** 14+
3. **微信开发者工具**
4. **Git** 版本控制

### 开发流程
```bash
# 1. 克隆项目
git clone <repository>

# 2. 安装依赖
npm install

# 3. 启动Cocos Creator
cocos creator

# 4. 构建微信小游戏
cocos build --platform wechatgame

# 5. 打开微信开发者工具
open /Applications/wechatwebdevtools.app
```

## 🚀 快速启动指南

### 1. 环境准备
```bash
# 安装Cocos Creator
npm install -g cocos-creator
```

### 2. 打开项目
```bash
cd 微恐咖啡厅
cocos creator --project .
```

### 3. 开发调试
- 使用Chrome调试TypeScript
- 使用微信开发者工具真机调试
- 使用性能面板监控资源

```bash

```

## 🛡️ 代码规范

### TypeScript规范
1. **命名**：使用小驼峰命名法
2. **类型**：显式类型声明，避免any
3. **注释**：重要函数和类必须注释
4. **错误处理**：try-catch处理异常

### 架构规范
1. **单一职责**：每个类/函数只做一件事
2. **依赖注入**：使用依赖注入管理
3. **事件驱动**：使用事件系统解耦
4. **状态管理**：集中式状态管理
