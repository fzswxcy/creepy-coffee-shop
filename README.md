# 微恐咖啡厅 (Creepy Coffee Shop)

一款独特的微信小游戏，融合微恐氛围与咖啡厅经营玩法。

## 🎮 游戏简介

《微恐咖啡厅》是一款经营类微信小游戏，玩家扮演咖啡厅老板，在神秘诡异的氛围中经营自己的咖啡厅，为各种奇特的顾客提供服务。

## ✨ 核心特性

- **微恐氛围** - 独特的诡异咖啡厅场景设计
- **经营玩法** - 制作咖啡、服务顾客、管理店铺
- **角色系统** - 多种性格迥异的顾客角色
- **成就系统** - 丰富的游戏成就和收集要素
- **社交功能** - 微信好友排行榜和分享

## 🛠️ 技术栈

- **引擎**: Cocos Creator 3.x
- **语言**: TypeScript
- **平台**: 微信小游戏
- **状态管理**: 自定义事件系统
- **数据持久化**: 微信云存储

## 📁 项目结构

```
微恐咖啡厅/
├── assets/           # 游戏资源
├── pages/            # 页面文件
├── art-assets/       # 美术资源
├── backend/          # 后端服务
└── weixin-game/      # 微信游戏版本
```

## 🚀 快速开始

1. 使用微信开发者工具打开项目
2. 导入 `微恐咖啡厅/` 或 `微信小游戏-微恐咖啡厅/` 目录
3. 点击编译运行
