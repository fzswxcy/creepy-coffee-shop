# 微信云开发环境配置指南

## 1. 环境准备

### 1.1 开通微信云开发
1. 访问 [微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html) 下载最新版本
2. 使用微信小程序账号登录
3. 创建小程序项目（选择"小游戏"类型）
4. 在开发者工具中开通"云开发"服务

### 1.2 创建云开发环境
- **环境名称**: `weikong-coffee-{env}` (如: weikong-coffee-dev, weikong-coffee-prod)
- **环境ID**: 系统自动生成
- **数据库**: 默认开启
- **云函数**: 默认开启
- **云存储**: 默认开启

## 2. 环境配置

### 2.1 环境变量配置
在 `miniprogram/app.js` 中初始化：

```javascript
// app.js
App({
  onLaunch: function () {
    if (!wx.cloud) {
      console.error('请使用 2.2.3 或以上的基础库以使用云能力')
    } else {
      wx.cloud.init({
        // 默认环境
        env: 'weikong-coffee-dev',
        // 动态切换环境
        traceUser: true,
      })
    }
  }
})
```

### 2.2 多环境配置
创建环境配置文件 `config/environment.js`:

```javascript
// config/environment.js
const environments = {
  dev: {
    env: 'weikong-coffee-dev',
    version: '1.0.0-dev',
    debug: true
  },
  staging: {
    env: 'weikong-coffee-staging',
    version: '1.0.0-staging',
    debug: true
  },
  prod: {
    env: 'weikong-coffee-prod',
    version: '1.0.0',
    debug: false
  }
}

// 根据构建模式选择环境
const currentEnv = (() => {
  if (process.env.NODE_ENV === 'production') {
    return environments.prod
  } else if (process.env.NODE_ENV === 'staging') {
    return environments.staging
  } else {
    return environments.dev
  }
})()

module.exports = currentEnv
```

## 3. 权限设置

### 3.1 数据库权限
在微信开发者工具中设置数据库安全规则：

```json
{
  "users": {
    ".read": true,
    ".write": "auth != null"
  },
  "game_data": {
    ".read": "auth != null",
    ".write": "auth != null"
  },
  "orders": {
    ".read": "auth != null",
    ".write": "auth != null"
  },
  "leaderboard": {
    ".read": true,
    ".write": "auth != null"
  }
}
```

### 3.2 云函数权限
所有云函数默认运行在管理员权限下，可访问所有数据。

## 4. 成本控制策略

### 4.1 数据库优化
- **按量计费**: 使用按量计费模式，不预购资源包
- **索引优化**: 为常用查询字段创建索引
- **分页查询**: 所有列表查询使用分页，限制每次查询数量
- **字段压缩**: 使用短字段名，避免冗余数据

### 4.2 云函数优化
- **内存配置**: 使用最小配置（128MB）
- **超时设置**: 设置合理超时时间（3-5秒）
- **冷启动优化**: 使用全局变量缓存常用数据
- **并发控制**: 监控云函数调用频率

### 4.3 云存储优化
- **资源压缩**: 图片、音频资源使用压缩格式
- **CDN缓存**: 启用CDN缓存，设置合理缓存时间
- **按需加载**: 游戏资源分批次加载

## 5. 部署流程

### 5.1 开发环境部署
```bash
# 1. 安装依赖
npm install

# 2. 上传云函数
tcb fn deploy --env weikong-coffee-dev

# 3. 初始化数据库
node scripts/init-database.js --env dev
```

### 5.2 生产环境部署
```bash
# 1. 构建项目
npm run build

# 2. 上传云函数
tcb fn deploy --env weikong-coffee-prod

# 3. 同步生产数据
node scripts/sync-production-data.js
```

## 6. 监控与告警

### 6.1 云开发控制台监控
- **数据库读写次数**: 每日监控
- **云函数调用次数**: 实时监控
- **云存储流量**: 周度分析
- **错误率**: 异常告警

class Monitor {

## 7. 安全配置

### 7.1 数据安全
- **用户数据隔离**: 每个用户只能访问自己的数据
- **敏感信息加密**: 使用微信提供的加密接口
- **API调用限制**: 限制高频调用

### 7.2 隐私合规
- **用户协议**: 明确数据收集和使用说明
- **隐私政策**: 符合微信小程序规范
- **数据删除**: 提供用户数据删除功能

## 8. 备份策略

### 8.1 数据库备份
- **频率**: 每日自动备份
- **保留时间**: 30天
- **存储位置**: 云存储备份目录

### 8.2 云函数备份
- **版本管理**: 每次部署保存版本
- **回滚机制**: 支持快速回滚到前一版本

## 快速开始

1. **初始化项目**
```bash
cd 微恐咖啡厅/backend
npm install
```

2. **配置环境**
```bash
cp .env.example .env
# 编辑.env文件，填入微信云开发环境ID
```

3. **部署开发环境**
```bash
npm run deploy:dev
```

4. **验证部署**
```bash
npm run verify:dev
```

## 注意事项

1. **成本控制**: 开发阶段使用免费额度，上线前评估成本
2. **性能优化**: 关注数据库查询性能，避免全表扫描
3. **安全合规**: 定期审查数据访问权限和隐私政策
4. **监控告警**: 设置成本告警，避免意外费用
