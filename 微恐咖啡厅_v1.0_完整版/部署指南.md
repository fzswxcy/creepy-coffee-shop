# 🚀 微恐咖啡厅 v1.0 部署指南

## 📋 部署前准备

### 1. 环境要求
- **微信开发者工具**: [下载地址](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)
- **微信小程序账号**: 需要注册并获取AppID
- **操作系统**: Windows/macOS/Linux均可

### 2. 获取AppID
1. 访问 [微信公众平台](https://mp.weixin.qq.com/)
2. 注册小程序账号（如已注册请跳过）
3. 在"开发-开发管理"中找到你的AppID

### 3. 项目目录结构确认
确认你拥有以下目录结构：
```
微恐咖啡厅_v1.0_完整版/
├── 微信小游戏/           # 微信小程序项目
├── 部署指南.md          # 本文件
├── README.md           # 项目说明
└── 微恐咖啡厅v1.0_test.html # 浏览器测试页面
```

## 🛠️ 第一步：导入项目到微信开发者工具

### 方法一：直接导入（推荐）
1. 打开微信开发者工具
2. 点击左上角"项目" → "导入项目"
3. 选择 `微恐咖啡厅_v1.0_完整版/微信小游戏` 文件夹
4. 输入你的AppID
5. 项目名称填写"微恐咖啡厅"
6. 点击"导入"

### 方法二：新建项目后复制
1. 在微信开发者工具中新建项目
2. 项目类型选择"小程序"
3. 填写AppID和项目名称
4. 删除默认生成的 `pages/index` 目录
5. 将本项目的所有文件复制到新建的项目目录中

## 🧪 第二步：本地测试

### 1. 编译运行
1. 导入成功后，点击"编译"按钮（或按Ctrl+B/Cmd+B）
2. 等待编译完成（约3-5秒）
3. 查看模拟器中的游戏界面

### 2. 功能测试清单
✅ **基础功能测试**：
- [ ] 页面正常加载，无错误提示
- [ ] 状态栏显示正常（资金、声誉、时间）
- [ ] 咖啡制作按钮可点击
- [ ] 进度条动画正常显示

✅ **游戏逻辑测试**：
- [ ] 制作咖啡能增加资金
- [ ] 顾客出现倒计时正常
- [ ] 服务顾客增加资金和声誉
- [ ] 时间流逝效果正常
- [ ] 每日结束时结算正常

✅ **用户体验测试**：
- [ ] 界面布局在不同分辨率下正常
- [ ] 点击响应及时，无延迟
- [ ] 游戏日志信息清晰
- [ ] 按钮状态变化明显

### 3. 开发者工具调试
1. **Console面板**: 查看游戏运行日志
2. **Sources面板**: 调试JavaScript代码
3. **Network面板**: 查看网络请求（本项目无网络请求）
4. **Storage面板**: 查看本地存储数据

## 📱 第三步：真机预览

### 1. 生成预览二维码
1. 在开发者工具中点击"预览"按钮
2. 等待编译完成
3. 使用微信扫描生成的二维码

### 2. 真机测试注意事项
- **网络环境**: 确保手机连接网络
- **微信版本**: 确保微信版本在8.0以上
- **权限设置**: 首次访问可能需要授权
- **设备兼容性**: 在不同手机型号上测试

### 3. 真机测试清单
✅ **功能验证**：
- [ ] 游戏界面显示完整
- [ ] 触摸操作响应准确
- [ ] 音频和震动反馈（如果启用）
- [ ] 横竖屏适配正常

✅ **性能测试**：
- [ ] 游戏运行流畅（30FPS+）
- [ ] 内存占用正常（<100MB）
- [ ] 电池消耗在合理范围
- [ ] 无闪退或卡死现象

## 🔧 第四步：配置修改

### 1. AppID配置
打开 `project.config.json` 文件，修改以下字段：
```json
{
  "appid": "你的AppID",
  "projectname": "微恐咖啡厅"
}
```

### 2. 游戏参数调整（可选）
在 `app.js` 中可调整游戏参数：
```javascript
// 初始游戏数据
this.gameData = {
  money: 100,           // 初始资金（可调整）
  reputation: 0,        // 初始声誉
  day: 1,               // 初始天数
  time: '09:00',        // 开始时间
  customersServed: 0,
  coffeeMade: 0,
  isOpen: true,
};
```

### 3. UI样式调整（可选）
在 `pages/game/game.wxss` 中可调整样式：
- 颜色方案
- 字体大小
- 按钮样式
- 布局间距

## 🐛 常见问题解决

### Q1: 导入项目时报错"找不到app.json"
**解决方案**：
- 检查项目目录结构是否正确
- 确保 `app.json` 文件存在
- 检查文件编码是否为UTF-8

### Q2: 真机预览时黑屏或白屏
**解决方案**：
- 检查网络连接
- 清除微信缓存
- 重启微信开发者工具
- 检查代码是否有语法错误

### Q3: 游戏性能不佳
**优化建议**：
- 减少不必要的DOM操作
- 优化图片资源大小
- 使用setData批量更新
- 避免频繁的定时器

### Q4: 审核被拒
**应对策略**：
- 仔细阅读审核拒绝理由
- 针对性地修改内容
- 重新提交审核
- 如遇技术问题，可联系微信客服
