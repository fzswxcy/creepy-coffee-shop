# 🎮 微恐咖啡厅 v1.0 完整版

## 📱 项目概述

**《微恐咖啡厅》**是一款融合模拟经营与微恐元素的微信小游戏。玩家经营一家独特的咖啡厅，制作咖啡服务顾客，同时体验微妙的恐怖氛围。

## 🎮 游戏机制详解

### 1. 经济系统
- **初始资金**: 100元
- **咖啡收入**: 美式15元、拿铁20元、卡布奇诺22元
- **服务收入**: 每服务一位顾客+25元
- **日常成本**: 每天结束-50元运营成本
- **声誉系统**: 成功服务+1，顾客离开-2

### 2. 时间系统
- **现实1秒** ≈ 游戏10分钟
- **营业时间**: 09:00 - 21:00
- **每日循环**: 每天结束后自动进入下一天
- **顾客生成**: 每小时有一定概率生成新顾客

### 3. 核心玩法
1. **制作咖啡**: 点击咖啡类型开始制作（带进度条）
2. **服务顾客**: 顾客出现后需在30秒内点击服务按钮
3. **经营决策**: 合理安排咖啡制作与顾客服务的时间
4. **长期发展**: 积累资金，提升声誉，持续经营

## 🛠️ 技术架构

### 前端技术栈
- **微信小程序框架**: WXML/WXSS/JavaScript
- **游戏架构**: 事件驱动 + 状态管理
- **UI框架**: 原生小程序组件 + 自定义样式
- **数据管理**: 全局App数据 + 本地存储

### 性能优化
- **首屏加载**: 资源最小化，基础包<500KB
- **渲染优化**: Canvas分层渲染，保证流畅体验
- **内存管理**: 定时器清理，避免内存泄漏
- **网络请求**: 本地模拟数据，无需服务器依赖

## 🐛 已知问题和解决方案

### 1. 微信审核问题
- **问题**: 微恐元素可能影响审核
- **解决方案**: PG-13级设计，已设置年龄分级

### 2. 性能问题
- **问题**: 低端设备可能卡顿
- **解决方案**: 自适应渲染质量，保证30FPS底线

### 3. 变现系统
- **问题**: 微信广告组件集成复杂
- **解决方案**: 分层变现设计，模拟广告兜底
