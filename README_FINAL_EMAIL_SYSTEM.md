
#### **第2步：更新配置文件**
编辑 `email_config.ini`，修改一行：
```ini

```

#### **第3步：运行系统**
```bash

# 读取和处理邮件（主功能）
```


## 🛠️ 故障排除

### **常见问题1：连接失败**
**症状**：`登录失败: b'Login fail...'`
**解决**：
1. 确认授权码正确（不是密码）
2. 检查QQ邮箱是否已开启IMAP服务
3. 尝试网页登录QQ邮箱验证

### **常见问题2：附件保存失败**
**症状**：`权限被拒绝`或`磁盘空间不足`
**解决**：
1. 检查附件目录权限：`ls -ld /tmp/Email_Attachments`
2. 检查磁盘空间：`df -h`
3. 修改配置文件中的附件保存目录

### **常见问题3：发送失败**
**症状**：`SMTPAuthenticationError`
**解决**：
1. 确认SMTP服务已开启（与IMAP一起开启）
2. 检查网络连接和防火墙设置
3. 确认发送频率未超限
