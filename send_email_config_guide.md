## 🔧 配置文件更新

编辑 `email_config.ini`，添加SMTP配置：

```ini
[SERVER]
; IMAP服务器配置
imap_host = imap.qq.com
imap_port = 993
use_ssl = true

; SMTP服务器配置（发送邮件）
smtp_host = smtp.qq.com
smtp_port = 465
smtp_use_ssl = true

[ACCOUNT]
; 邮箱账号信息
email_account = 731073066@qq.com
email_password = 你的16位授权码

[SENDER]
; 发件人显示信息（可选）
display_name = NIKO助手

[SAVE]
; 附件保存目录
attachment_dir = /tmp/Email_Attachments/

[OPTIONS]
; 处理选项
mark_as_read = false
preview_length = 200
```

## 🛠️ 使用指南

### 1. **发送测试邮件**
```bash
python3 send_email.py
```
选择选项1，系统会自动发送测试邮件给你自己。

### 2. **单封邮件发送**
选择选项2，按照提示：
1. 输入收件人地址
2. 填写邮件主题
3. 输入邮件正文
4. 添加附件（可选）
5. 确认发送

### 3. **批量邮件发送**
```bash
# 先创建收件人列表文件
echo "user1@example.com" > recipients.txt
echo "user2@example.com" >> recipients.txt

# 运行批量发送
python3 send_email.py
选择选项3，指定收件人文件
```

## 🚨 常见问题解决

### Q1: 发送失败（认证错误）
**症状**：`SMTPAuthenticationError`
**解决**：
1. 确认授权码正确
2. 检查SMTP服务是否开启
3. 确认邮箱安全设置

### Q2: 发送失败（连接超时）
**症状**：`SMTPConnectError`
**解决**：
1. 检查网络连接
2. 验证SMTP服务器地址和端口
3. 检查防火墙设置

### Q3: 邮件被拒收
**症状**：邮件发送成功但被退回
**解决**：
1. 检查发件人地址合法性
2. 避免垃圾邮件关键词
3. 控制发送频率
